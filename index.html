Тренировка Питание

Тренировка

День 1: Грудь и трицепсы

  • Жим штанги лежа: 4 подхода по 6-8 повторений
  • Жим гантелей на наклонной скамье: 3 подхода по 8-10 повторений
  • Разведения гантелей: 3 подхода по 10-12 повторений
  • Жим узким хватом: 3 подхода по 8-10 повторений
  • Французский жим: 3 подхода по 10-12 повторений

День 2: Спина и бицепсы

  • Подтягивания: 4 подхода по максимальному количеству повторений
  • Тяга штанги в наклоне: 3 подхода по 8-10 повторений
  • Тяга гантели одной рукой: 3 подхода по 10-12 повторений
  • Тяга Т-грифа: 3 подхода по 10-12 повторений
  • Сгибания штанги на бицепс: 3 подхода по 8-10 повторений
  • Сгибания гантелей на бицепс: 3 подхода по 10-12 повторений

День 3: Отдых

День 4: Ноги

  • Приседания со штангой: 4 подхода по 6-8 повторений
  • Жим ногами: 3 подхода по 8-10 повторений
  • Выпады с гантелями: 3 подхода по 10-12 повторений
  • Разгибания ног в тренажере: 3 подхода по 12-15 повторений
  • Сгибания ног в тренажере: 3 подхода по 12-15 повторений

День 5: Плечи и трапеции

  • Жим штанги над головой: 4 подхода по 6-8 повторений
  • Жим гантелей сидя: 3 подхода по 8-10 повторений
  • Разведения гантелей в стороны: 3 подхода по 10-12 повторений
  • Шраги с гантелями: 3 подхода по 10-12 повторений

День 6: Отдых

День 7: Повторение одного из дней по выбору или отдых

Питание

Завтрак

  • Овсянка на молоке с ягодами и орехами
  • Омлет из 4 яиц

Перекус

  • Протеиновый коктейль
  • Банан

Обед

  • Куриная грудка
  • Коричневый рис
  • Овощи на пару

Перекус

  • Творог с медом
  • Орехи

Ужин

  • Лосось или другая жирная рыба
  • Киноа или сладкий картофель
  • Салат из свежих овощей

Перекус перед сном

  • Греческий йогурт
  • Миндаль
